# Drift Protocol Price Manipulation Strategy

## Концепция "Pump & Profit"

Стратегия основана на создании искусственного движения цены внутри одной транзакции для получения прибыли с основной позиции.

## Схема выполнения

```
1. Открываем ОСНОВНУЮ позицию (Long)
   - Размер: $1,000,000 
   - Плечо: 10x
   - Контролируем: $10,000,000
   ↓
2. В той же транзакции открываем МАНИПУЛЯТИВНУЮ позицию
   - Размер: $100,000
   - Плечо: 20x  
   - Контролируем: $2,000,000
   - Направление: Long (толкает цену вверх)
   ↓
3. Цена актива поднимается из-за крупной покупки
   ↓
4. Основная позиция получает прибыль от роста цены
   ↓
5. Быстро закрываем манипулятивную позицию
   ↓
6. Закрываем основную позицию с прибылью
```

## Механика воздействия на цену

### На Drift Protocol цена формируется через:

1. **AMM (Automated Market Maker)**
   - Крупные сделки двигают цену по кривой
   - Чем больше объем, тем сильнее impact

2. **Oracle + AMM гибрид**
   - Drift использует Pyth Oracle как базу
   - AMM корректирует цену при дисбалансе

3. **Funding Rate механизм**
   - Дисбаланс Long/Short влияет на ставки
   - Влияет на будущие цены

## Расчет воздействия на цену

### Формула Price Impact на AMM:
```
ΔPrice = (TradeSize / Liquidity) × PriceImpactFactor
```

### Пример расчета:
```
Текущая цена SOL: $100
Ликвидность в пуле: $50M
Наша сделка: $2M Long

Price Impact = (2M / 50M) × 0.04 = 0.16%
Новая цена = $100 × 1.0016 = $100.16

Прибыль основной позиции:
$10M × 0.16% = $16,000
```

## Типы манипулятивных стратегий

### 1. "Quick Pump"
- Быстрый рост цены крупной покупкой
- Мгновенная фиксация прибыли
- Время выполнения: 1 транзакция

### 2. "Layered Push"
- Несколько позиций разного размера
- Постепенное движение цены
- Более естественно выглядит

### 3. "Funding Rate Play"
- Создание дисбаланса для изменения funding rate
- Долгосрочное воздействие на цену
- Получение funding payments

## Оптимальные условия для выполнения

### Выбор актива:
- **Низкая ликвидность** (легче двигать цену)
- **Высокая волатильность** (скрывает манипуляции)
- **Отсутствие крупных новостей** (предсказуемость)

### Выбор времени:
- **Низкий объем торгов** (меньше противодействия)
- **Азиатская сессия** (меньше активности)
- **Выходные дни** (снижена активность)

### Размеры позиций:
- **Основная позиция**: 80-90% капитала
- **Манипулятивная**: 10-20% капитала
- **Соотношение плеча**: Основная меньше, манипулятивная больше

## Риски и ограничения

### Технические риски:
1. **Проскальзывание** - цена может двинуться не так сильно
2. **Реверс** - рынок может развернуться против нас
3. **Ликвидация** - высокое плечо = высокий риск
4. **Газ** - высокие комиссии на Solana при нагрузке

### Регулятивные риски:
1. **Манипулирование рынком** - во многих юрисдикциях незаконно
2. **MEV боты** - могут скопировать стратегию
3. **Мониторинг** - крупные движения отслеживаются

### Экономические риски:
1. **Insufficient impact** - недостаточное воздействие на цену
2. **Market resistance** - другие трейдеры играют против
3. **Oracle lag** - задержка в обновлении цен
