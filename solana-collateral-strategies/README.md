# Solana Collateral Strategies 🏦

## Протоколы кредитования на Solana

### 1. Solend
- **TVL**: $150M+
- **Особенности**: 
  - Изолированные пулы для разных рисков
  - Flash loans с 0.003% комиссией
  - Поддержка LST (liquid staking tokens)
- **Залоги**: SOL, USDC, mSOL, stSOL, BTC, ETH

### 2. Kamino Finance
- **TVL**: $1.2B+
- **Особенности**:
  - kTokens - токенизированные CLMM позиции
  - Multiply (встроенный leverage)
  - Auto-compounding
- **Залоги**: kTokens, SOL, USDC, основные токены

### 3. MarginFi
- **TVL**: $500M+
- **Особенности**:
  - Высокие LTV (до 95% на стейблы)
  - Поддержка LST позиций
  - Интеграция с Jupiter
- **Залоги**: SOL, LST, стейблкоины, основные токены

### 4. Drift Protocol
- **TVL**: $200M+
- **Особенности**:
  - Перпетуальные фьючерсы
  - Spot margin trading
  - Cross-collateral
- **Залоги**: USDC, SOL для маржин трейдинга

### 5. Mango Markets
- **TVL**: $50M+
- **Особенности**:
  - Унифицированный маржин аккаунт
  - Perps + Spot в одном месте
  - Cross-margining
- **Залоги**: Все листинговые токены

## Типы залоговых активов

### 1. Обычные токены
- SOL, USDC, USDT
- Wrapped BTC/ETH
- Основные токены (BONK, JTO, JUP)

### 2. LST (Liquid Staking Tokens)
- mSOL (Marinade)
- stSOL (Lido)
- jitoSOL (Jito)
- **Преимущество**: Получаете staking rewards + можете использовать как залог

### 3. LP токены
- Orca LP
- Raydium LP
- **Проблема**: Обычно не принимаются напрямую

### 4. Токенизированные позиции
- kTokens (Kamino)
- Francium positions
- Tulip vaults
- **Преимущество**: Превращают LP в fungible токены

### 5. RWA (Real World Assets)
- USDY (Ondo)
- Различные токенизированные активы
- **Особенность**: Стабильная доходность
